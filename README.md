WordPress on LEMPER
===================

easily install WordPress on LEMP Stack.


### Usage

if WordPress On LEMPER is not cloned

    git clone git://github.com/buchin/wp-on-lemper.git

Then

    cd WordPress-On-Lemper
    sh wp.sh

and follow the tasks
